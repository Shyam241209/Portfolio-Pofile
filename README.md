# Portfolio Website

My personal portfolio website built using React, TypeScript, Tailwind CSS, and Vite.

## Features
- Responsive Design
- About Me
- Skills Section
- Projects Showcase
- Contact Form with Email Support
- Modern UI

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite
- Node.js
- Express.js
- Nodemailer

## Installation

```bash
npm install
npm run dev
```
